Telegram Notifications
======================

Purpose
-------

Use Telegram when Power Snitch should send alert messages through a bot to one or more specific chats.

Service-level settings
----------------------

Telegram services currently store:

- bot token

Channel-level settings
----------------------

Telegram channels currently store:

- chat ID
- optional extra message text

Operational notes
-----------------

- The bot must already exist and be able to send messages to the target chat.
- Confirm the chat ID independently before testing from Power Snitch.
