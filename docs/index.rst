Power Snitch Documentation
==========================

Power Snitch is a lightweight UPS monitoring and alerting appliance for Debian-family Linux systems. It combines local NUT integration, a FastAPI-based web application, SQLite-backed configuration, and multi-channel alerting in a single installable service.
